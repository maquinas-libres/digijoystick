Adaptadores de Joystick antiguos a USB para Attiny85

Requiere
========

* 2 Diodos zener 3V6
* 1 Resistencia 1,5KΩ 1/4W 
* 2 Resistencia 68Ω 1/4W
* 10 Resistencias de 10KΩ (para sega)
* 1 CD4021 (para sega)
* DB9 o DB15 macho
* puerto USB macho
